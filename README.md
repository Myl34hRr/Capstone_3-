# Bookstore Sales Analytics Dashboard | Power BI Capstone

## Project Overview

This capstone project analyzes bookstore sales data using Power BI to uncover business insights and support data-driven decision-making. The dashboard combines sales, product, store, and management data to identify sales trends, top-performing books, regional performance, and management assignments.

The project demonstrates the complete analytics workflow, including data cleaning, data modeling, DAX calculations, and interactive dashboard design.

---

## Business Problem

A bookstore company wanted to better understand its sales performance across multiple stores and regions. The goal was to identify:

* Top-selling books
* Revenue generated by each book
* Sales performance by region and state
* Store manager assignments
* Regional director responsibilities
* Overall sales trends

## Data Cleaning

The data was cleaned using Power Query by:

* Removing null values
* Standardizing state names
* Cleaning book titles
* Separating book titles from author names
* Removing extra spaces and special characters
* Correcting inconsistent data types
* Creating lookup tables
* Merging tables for improved reporting

---

## Data Model

The project includes relationships between:

* Store Sales
* Products
* Book List
* Store Locations
* State & Region
* State & Store Manager
* Region & Director

These relationships allow users to analyze sales from multiple perspectives.

---

## Dashboard Features

The dashboard includes:

* Sales by Region
* Sales by State
* Top Selling Books
* Revenue by Book
* Author Information
* Store Manager Assignments
* Regional Director Assignments
* Interactive slicers for:

  * Region
  * State
  * Product
  * Date

## Key Insights

* Identified the highest-selling books across all stores.
* Measured revenue generated by individual books.
* Compared sales performance by state and region.
* Connected management assignments to store performance.
* Created an interactive dashboard that allows users to filter results by region and state.
